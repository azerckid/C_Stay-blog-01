# 기능 명세: AI 여행 일지 모드 (AI Travel Log Mode)

이 문서는 사용자가 여행 현장에서 사진과 음성만으로 완성도 높은 여행기(트윗)를 작성할 수 있도록 돕는 'AI 여행 일지 모드'에 대한 기획 및 기술 명세를 담고 있습니다.

## 1. 주요 특징

*   **손쉬운 포착**: 텍스트 입력의 번거로움 없이 촬영과 음성 녹음만으로 기록을 완성합니다.
*   **멀티모달 분석**: AI가 사진의 시각적 요소와 사용자의 목소리를 결합하여 맥락 있는 문장을 생성합니다.
*   **개인화된 문체**: 사용자가 선호하는 여행기 스타일을 선택하여 맞춤형 결과를 얻을 수 있습니다.

## 2. 사용자 시나리오 (UX)

1.  **진입**: 메인 화면 또는 작성 창에서 '여행 일지 모드' 버튼을 통해 전용 UI로 진입합니다.
2.  **촬영**: 앱 내장 카메라를 통해 현장 사진을 촬영합니다.
3.  **음성 기록**: '말하기' 버튼을 눌러 당시의 느낌, 장소, 상황 등을 자유롭게 이야기합니다.
4.  **스타일 선택**: 제공되는 문체 옵션 중 하나를 선택합니다.
    *   *감성적*: "노을빛이 스며드는 이 순간..."
    *   *정보 전달*: "현재 에펠탑 인근의 날씨는..."
    *   *위트/발랄*: "와! 진짜 대박! 이거 보러 파리 왔지..."
5.  **AI 생성**:
    *   사용자의 투박한 음성 인식 결과 (STT)
    *   **GPS 좌표 기반 POI(관심 지점) 검색 (예: 카페명, 식당명 등 구체적 상호명 추출)**
    *   사진 메타데이터(EXIF)의 위치 정보 또는 실시간 기기 GPS 활용
    *   사진을 분석하여 시각적 특징 추출 (Gemini Vision)
    *   선택한 문체에 맞춰 위 정보들을 조합한 여행기 초안 생성 (예: "경주의 아담한 카페 'OO'에서 맛본 달콤한 디저트...")
6.  **확인 및 발행**: 생성된 문장을 확인하고 필요시 수정한 뒤 즉시 피드에 게시합니다.

## 3. 기술적 상세

### AI 모델 및 통합
*   **비전 분석**: `Gemini 1.5 Flash` (이미지 내 장소, 사물, 분위기 파악)
*   **문항 생성**: `Gemini 1.5 Pro/Flash` (STT 결과 + 비전 분석 데이터 + 사용자 선택 문체)
*   **STT (Speech-to-Text)**: OpenAI Whisper API 또는 Google Cloud STT 전용 어댑터 활용

### 데이터 구조 (Tweet 확장)
*   생성된 트윗에는 AI가 분석한 위치 정보와 태그가 자동으로 추천되어 포함됩니다.
*   Metadata 필드에 `generationMode: 'AI_VOICE_LOG'` 정보가 기록됩니다.

### 기기 연동 (Capacitor)
*   `@capacitor/camera`: 앱 내 고화질 촬영 및 갤러리 연동
*   `@capacitor/voice-recorder`: 고음질 음성 녹음 및 서버 전송용 파일 처리

## 4. 향후 확장성

*   **Vlog 연동**: 촬영된 사진과 녹음된 음성을 짧은 영상으로 자동 편집해주는 기능
*   **여행 지도 자동 생성**: 일지 모드로 작성된 게시물들만 모아 지도 위에 경로로 표시
